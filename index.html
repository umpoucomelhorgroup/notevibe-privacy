Política de Privacidade - NoteVibe

Última atualização: 10 de abril de 2026

1. Introdução

O aplicativo NoteVibe respeita a sua privacidade e se compromete a proteger quaisquer dados do usuário conforme a LGPD (Lei Geral de Proteção de Dados - Lei nº 13.709/2018).

Ao utilizar o app, você concorda com esta Política de Privacidade.

2. Dados armazenados

O NoteVibe armazena dados exclusivamente no dispositivo do usuário.

  • Notas criadas pelo usuário
  • Títulos e conteúdos das notas
  • Configurações do aplicativo
  • Backups gerados pelo usuário

Não enviamos dados automaticamente para servidores externos.

3. Armazenamento local (SQLite)

As notas são armazenadas localmente usando banco de dados SQLite no dispositivo.

4. Backup criptografado

O aplicativo permite criar backups locais protegidos por senha.

  • Criptografia com senha definida pelo usuário
  • Armazenamento local ou pasta escolhida pelo usuário
  • Sem envio automático para a internet

5. Permissões

O app pode solicitar acesso ao armazenamento apenas para salvar backups.

6. Segurança

Utilizamos armazenamento local e criptografia para proteger os dados. Porém, o usuário também é responsável pela segurança do seu dispositivo e senha.

7. Responsabilidade do usuário

  • Manter a senha de backup segura
  • Proteger o acesso ao dispositivo
  • Gerenciar seus próprios arquivos de backup

8. Direitos do usuário (LGPD)

O usuário pode excluir seus dados removendo o aplicativo ou apagando as notas manualmente.

9. Contato

Para dúvidas, entre em contato com o desenvolvedor do NoteVibe.
