Política de Privacidade - NoteVibe - Notas e listas

Última atualização: 29 de abril de 2026

1. Introdução

O aplicativo NoteVibe - Notas e listas é desenvolvido por WPrimeApps. Respeitamos sua privacidade e nos comprometemos a proteger seus dados conforme a LGPD (Lei nº 13.709/2018) e outras leis aplicáveis de proteção de dados.

2. Coleta de Dados

O NoteVibe não coleta, armazena em servidores externos ou compartilha dados pessoais do usuário. Todos os dados inseridos no aplicativo permanecem exclusivamente no dispositivo do usuário.

3. Dados Armazenados Localmente

Para o funcionamento das notas, as seguintes informações são armazenadas no banco de dados local (SQLite) do seu dispositivo:

  • Títulos e conteúdos das notas e listas;
  • Configurações de preferência do aplicativo;
  • Arquivos de backup gerados manualmente pelo usuário.

4. Armazenamento e Backup

Os dados são processados localmente. O aplicativo oferece uma função de backup criptografado para segurança do usuário:

  • Local de Destino: Ao gerar um backup, o arquivo é salvo automaticamente na pasta de Downloads do dispositivo.
  • Criptografia: Os backups são protegidos por uma senha definida pelo usuário. A WPrimeApps não tem acesso a essa senha; a perda da mesma impossibilita a recuperação dos dados.
  • Sincronização: Não há envio automático de dados para a nuvem ou internet.

5. Permissões

O aplicativo solicita permissões estritamente necessárias para suas funcionalidades:

  • Armazenamento / Mídia: Utilizado para salvar backups na pasta de Downloads, exportar arquivos (PDF/TXT) e permitir a seleção de imagens ou arquivos pelo usuário. Em versões mais antigas do Android, o sistema pode solicitar permissão de acesso ao armazenamento. Em versões mais recentes, o acesso é restrito e ocorre apenas mediante ação do usuário, utilizando os mecanismos seguros do sistema.
  • Notificações: Utilizadas para exibir lembretes criados pelo usuário.
  • Alarmes e Lembretes: Utilizados para agendar notificações nos horários definidos pelo usuário.

Imagens ou arquivos selecionados pelo usuário são utilizados apenas dentro do aplicativo e não são transmitidos para fora do dispositivo.

6. Segurança

Embora utilizemos criptografia nos backups e armazenamento local seguro, a segurança física e digital do dispositivo (senhas de bloqueio, biometria) é de responsabilidade do usuário.

7. Direitos do Usuário

Como o controle dos dados é totalmente local, o usuário exerce seus direitos de exclusão simplesmente apagando as notas dentro do app ou desinstalando o aplicativo, o que removerá o banco de dados local.

8. Contato

Para dúvidas sobre esta política, entre em contato através do e-mail: wprimeapps@gmail.com

Esta política pode ser atualizada periodicamente para refletir melhorias no aplicativo.

Privacy Policy - NoteVibe - Notas e listas

Last updated: April 29, 2026

1. Introduction

The NoteVibe - Notas e listas application is developed by WPrimeApps. We respect your privacy and are committed to protecting your data in accordance with applicable data protection laws.

2. Data Collection

NoteVibe does not collect, store on external servers, or share any personal user data. All data entered into the application remains exclusively on the user's device.

3. Locally Stored Data

To ensure the functionality of notes and lists, the following information is stored in the local database (SQLite) on your device:

  • Note and list titles and content;
  • Application preference settings;
  • Backup files manually generated by the user.

4. Storage and Backup

Data is processed locally. The app offers a password-encrypted backup function for user security:

  • Destination: When a backup is generated, the file is automatically saved to the device's Downloads folder.
  • Encryption: Backups are protected by a user-defined password. WPrimeApps has no access to this password; losing it will make data recovery impossible.
  • Synchronization: There is no automatic uploading of data to the cloud or the internet.

5. Permissions

The app requests permissions strictly necessary for its features:

  • Storage / Media: Used to save backups to the Downloads folder, export files (PDF/TXT), and allow the user to select images or files. On older Android versions, the system may request storage access permission. On newer versions, access is restricted and occurs only through user action, using secure system mechanisms.
  • Notifications: Used to display reminders created by the user.
  • Alarms and Reminders: Used to trigger notifications at the exact times defined by the user.

Images or files selected by the user are used only within the app and are not transmitted outside the device.

6. Security

While we use encryption for backups and secure local storage, the physical and digital security of the device (lock screen passwords, biometrics) is the user's responsibility.

7. User Rights

Since data control is entirely local, users exercise their right to deletion simply by deleting notes within the app or uninstalling the application, which will remove the local database.
