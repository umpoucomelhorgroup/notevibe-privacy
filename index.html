Política de Privacidade - NoteVibe

Última atualização: 18 de abril de 2026

1. Introdução

O NoteVibe respeita sua privacidade e se compromete a proteger seus dados conforme a LGPD (Lei nº 13.709/2018) e outras leis aplicáveis de proteção de dados.

2. Coleta de Dados

O NoteVibe não coleta, armazena ou compartilha dados pessoais fora do dispositivo do usuário.

3. Dados armazenados

  • Notas criadas pelo usuário
  • Títulos e conteúdos das notas
  • Configurações do aplicativo
  • Backups gerados pelo usuário

Todos os dados permanecem exclusivamente no dispositivo.

4. Armazenamento local

Os dados são armazenados localmente utilizando banco de dados SQLite.

5. Backup criptografado

O aplicativo permite criar backups protegidos por criptografia baseada em senha definida pelo usuário.

  • Armazenamento local ou em pasta escolhida
  • Nenhum envio automático para a internet

6. Permissões

O aplicativo pode solicitar as seguintes permissões:

  • Armazenamento / Mídia: Utilizado para salvar backups, exportar arquivos (PDF/TXT) e permitir a seleção de imagens. Em versões recentes do Android, o acesso é restrito apenas aos arquivos do app; em versões anteriores, o sistema poderá solicitar permissão explícita de acesso.
  • Notificações: utilizadas para exibir lembretes criados pelo usuário.
  • Alarmes e lembretes: utilizados para agendar notificações nos horários definidos pelo usuário.

As imagens selecionadas pelo usuário são utilizadas apenas dentro do aplicativo e permanecem armazenadas localmente no dispositivo.

Essas permissões são utilizadas exclusivamente para o funcionamento do aplicativo e não para coleta ou compartilhamento de dados.

7. Segurança

Utilizamos armazenamento local e criptografia para proteger os dados. O usuário é responsável pela segurança do dispositivo e da senha.

8. Direitos do usuário

O usuário pode excluir seus dados a qualquer momento removendo o aplicativo ou apagando as notas manualmente.

9. Contato

Email: wprimeapps@gmail.com

Recomendamos revisar esta política periodicamente.

Privacy Policy - NoteVibe

Last updated: April 18, 2026

1. Introduction

NoteVibe respects your privacy and is committed to protecting your data in accordance with LGPD (Law No. 13.709/2018) and other applicable data protection laws.

2. Data Collection

NoteVibe does not collect, store, or share personal data outside the user's device.

3. Stored Data

  • User-created notes
  • Titles and contents of notes
  • App settings
  • User-generated backups

All data remains exclusively on the device.

4. Local Storage

Data is stored locally using a SQLite database.

5. Encrypted Backup

The app allows creating backups protected by encryption based on a password defined by the user.

  • Local storage or in a chosen folder
  • No automatic sending to the internet

6. Permissions

The app may request the following permissions:

  • Storage / Media: Used to save backups, export files (PDF/TXT), and allow image selection. In recent Android versions, access is limited to app-specific files; in older versions, the system may request explicit access permission.
  • Notifications: used to display reminders created by the user.
  • Alarms and reminders: used to schedule notifications at times defined by the user.

Images selected by the user are used only within the app and remain stored locally on the device.

These permissions are used exclusively for app functionality and not for data collection or sharing.

7. Security

We use local storage and encryption to protect data. The user is responsible for the security of the device and password.

8. User Rights

The user can delete their data at any time by removing the app or manually deleting notes.
